# IT-project

Мій навчальний проект для освоєння Git, HTML, CSS та JavaScript.

## Що зроблено:
* **HTML:** Створено структуру сторінки із заголовком.
* **CSS:** Підключено зовнішні стилі, налаштовано колір фону та шрифти.
* **JavaScript:** Додано інтерактивність — при кліку на заголовок з'являється віконце `alert`.
* **Git:** Освоєно роботу з локальними та віддаленими гілками, а також злиття (`merge`).
